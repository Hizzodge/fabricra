# React Clothing Store

A practice e-commerce frontend built with **React, TypeScript, SCSS and Storybook**.
The project focuses on **component-driven development and UI architecture**.

## Tech Stack

- React
- TypeScript
- Vite
- SCSS
- Storybook

## Features

-Component-driven UI development

-Reusable UI components

-Storybook for isolated component development

-Modular SCSS styling

-Type-safe development with TypeScript

### Install dependencies

```bash
yarn
```

### Start development server

```bash
yarn dev
```

### Run Storybook

```bash
yarn storybook
```

## Build for production

```bash
yarn build
```
