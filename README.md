# React Clothing Store

A practice e-commerce frontend built with **React, TypeScript, SCSS and Storybook**.
The project focuses on **component-driven development and UI architecture**.

## Tech Stack

- React
- TypeScript
- Vite
- SCSS
- Storybook

## Getting Started

### Install dependencies

```bash
yarn
```

### Start development server

```bash
yarn dev
```

### Run Storybook

```bash
yarn storybook
```

## Build for production

```bash
yarn build
```
